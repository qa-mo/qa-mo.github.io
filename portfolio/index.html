How It All Connects

CI/CD Trigger

Push to uat or master branch triggers AWS CodePipeline. Scheduled runs via AWS Scheduler fire every weekday at 8 PM.

Parallel Execution

CodeBuild batch builds spin up 4 parallel workers. Cypress Cloud orchestrates spec distribution and records results centrally.

Reporting & Alerts

Mochawesome reports merge into a single HTML artifact. Results post to Slack and persist in DynamoDB via the Monocle Quality Dashboard API.

Multi-Environment

Same test suites run against UAT and Production. Environment-specific configs control base URLs, Cognito pools, and S3 buckets.
